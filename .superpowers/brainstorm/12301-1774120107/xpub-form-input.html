Ein Feld, auto-erkannt

Dasselbe Eingabefeld wie heute. Wenn der User einen zpub/xpub/ypub einfügt, wird es automatisch erkannt — genau wie normale Adressen.

Pro

  • Kein UI-Umbau nötig
  • Nahtlose UX
  • User muss nichts umschalten

Contra

  • Kein Hinweis dass xpub möglich ist

User wählt explizit zwischen "Adresse" und "xpub / HD-Wallet". Klarer Hinweis dass beide Eingabearten existieren.

Pro

  • User weiß dass xpub möglich ist
  • Klare Trennung der Modi

Contra

  • Mehr UI-Aufwand
  • Mehr Klicks
